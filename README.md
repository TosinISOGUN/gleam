# Gleam - Professional Cleaning Services in Ibadan

Gleam (Gleam Cleaning Co.) is a premium cleaning service provider based in Ibadan, Oyo State, Nigeria. We specialize in bringing a professional touch to homes and offices, ensuring every space we touch sparkles and shines!

## 🌟 Key Features

- **Home Cleaning**: Thorough cleaning of bedrooms, kitchens, bathrooms, and living areas.
- **Office Cleaning**: Professional maintenance for productive workspaces.
- **Deep Cleaning**: Intensive care for spaces that need extra attention.
- **Trained Professionals**: Vetted and experienced cleaning team.
- **Eco-Friendly**: We use professional-grade, environmentally safe cleaning products.

## 🛠️ Technology Stack

This project is built with a modern, performant web stack:

- **React 18** & **TypeScript**
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **Shadcn UI** (Components)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Local Development

1. **Clone the repository**:
   ```sh
   git clone <repository-url>
   cd gleam
   ```

2. **Install dependencies**:
   ```sh
   npm install
   # or
   bun install
   ```

3. **Start the development server**:
   ```sh
   npm run dev
   # or
   bun dev
   ```

4. **Open your browser**:
   Navigate to `http://localhost:5173` to see the application in action.

## 📖 Project Structure

- `src/pages`: Main application views (Index, About, Services, FAQ, Contact).
- `src/components`: Reusable UI components and layouts.
- `src/assets`: Visual assets and imagery.
- `src/hooks`: Custom React hooks for shared logic.
